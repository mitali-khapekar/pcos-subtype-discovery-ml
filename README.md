# PCOS Subtype Discovery using Machine Learning

## Overview
This project identifies hidden PCOS subtypes using Machine Learning techniques such as K-Means Clustering and Random Forest Classification.

## Features
- Lean PCOS and Metabolic PCOS prediction
- K-Means clustering
- PCA dimensionality reduction
- Silhouette Score evaluation
- Streamlit user interface

## Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit
- Google Colab

## Machine Learning Techniques
- K-Means Clustering
- PCA
- Random Forest Classifier
